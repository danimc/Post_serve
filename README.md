### Despliegue del sistema

  lo primero que debemos hacer es ejecutar npm-install para recuperar todas las dependencias de node
  
 ```

```

 para ejecutar el sistema hay que realizarlo desde la carpeta dist/
 ```
npm node dist/app
```
